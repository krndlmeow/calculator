#Техническое задание. Программа "Калькулятор".

#Цель: Разработать простое консольное приложение «Калькулятор», выполняющее основные арифметические операции над числами с учётом обработки ошибок ввода.

#Исполнитель: Grizoglazov M.E.

#Функциональные требования:
##-Пользователь должен иметь возможность вводить два числа и выбирать операцию

##-Программа должна поддерживать основные арифмитические операции: сложение
##вычитание, умножение, деление. И ряд дополнительных возможностей: возведение
##числа в степень и извлечение квадратного корня

##-Программа должна выводить результат вычисления

##-Программа должна обрабатывать ошибки по типу: деление на ноль, ввод слишком большого числа, ввод 
##пользователем не числа, ввод пользователем вещественного числа с точностью не превышающей два знака после запятой

#Технические требования:
##-Программа должна быть написана на языке C#

##-Программа должна работать в консоли

##-Программа должна быть простой и понятной для пользователя

#Алгоритм работы программы:
##1. Программа запрашивает у пользователя 2 числа.
##2. Программа запрашивает у пользователя требуемую операцию.
##3. Программа выполняет выбранную операцию над введенными числами.
##4. Программа обрабатывает ошибки, если они есть.
##5. Программа выводит результат.

#Тест-кейсы:

##1. Сложение(+)

| № | Ввод 1 | Ввод 2 | Операция | Ожидаемый вывод |
|:-:|:------:|:------:|:----------:|:----------------|
| 1 | 5 | 3 | + | Результат: 8 |
| 2 | -2 | 7 | + | Результат: 5 |
| 3 | 1.25 | 2.50 | + | Результат: 3.75 |
| 4 | abc | 2 | + | Ошибка: введите числовое значение. |
| 5 | 1.234 | 2 | + | Ошибка: допускается не более двух знаков после запятой. |
| 6 | 1E+400 | 2 | + | Ошибка: число слишком велико. |

| 2 | -3 | -7 | - | Результат: 4 |
| 3 | 2.50 | 1.25 | - | Результат: 1.25 |
| 4 | 2 | two | - | Ошибка: введите числовое значение. |
| 5 | 4.567 | 2 | - | Ошибка: допускается не более двух знаков после запятой. |
| 6 | 1E+400 | 3 | - | Ошибка: число слишком велико. |

| 2 | -4 | 5 | * | Результат: -20 |
